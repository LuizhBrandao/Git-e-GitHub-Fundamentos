# 📚 Git e GitHub - Fundamentos

Bem-vindo ao repositório **Git e GitHub - Fundamentos**. Este espaço foi desenhado para ser um roteiro de aprendizagem progressivo, focado no versionamento de código, abordando desde a teoria essencial até à publicação, colaboração e documentação de projetos. 

É um material de apoio ideal tanto para fins de ensino e orientação de novos programadores, quanto para guiar o desenvolvimento e apresentação de projetos de portfólio.

## 🗺️ Roteiro de Aprendizagem

### 1. 🟢 O Essencial: Teoria e Prática
Antes de executar comandos no terminal, é vital compreender como o versionamento funciona e qual é a relação entre o Git local e a nuvem.
* 📖 **[Documentação Oficial do GitHub](https://docs.github.com/pt)**: O manual de referência oficial para consultar sempre que houver dúvidas.
* 📄 **[Artigo: O que é Git e GitHub?](https://www.alura.com.br/artigos/o-que-e-git-github)**: Leitura fundamental para consolidar a diferença entre a ferramenta de versionamento e a plataforma de alojamento.
* 🎥 **[O que é Git e GitHub? (Definição e Conceitos)](https://www.youtube.com/watch?v=DqTITcMq68k)**: Introdução teórica focada nos conceitos principais: *commits*, *branches* e *merges*.
* 🎥 **[Como usar Git e GitHub na Prática!](https://www.youtube.com/watch?v=UBAX-13g8OM)**: Mãos à obra. O processo desde a instalação e o primeiro *commit*, até à criação de repositórios e *Pull Requests*.
* 🎧 **[Podcast Hipsters.Tech #184: Guia do Iniciante em GitHub](https://cursos.alura.com.br/extra/hipsterstech/guia-do-iniciante-em-github-hipsters-184-a378)**: Uma conversa excelente e orientada para quem está a dar os primeiros passos no versionamento.
* 🎧 **[Podcast Hipsters.Tech #109: Git e GitHub](https://www.alura.com.br/podcast/hipsterstech-git-e-github-hipsters-109-a474)**: Bate-papo aprofundado sobre os impactos da adoção destas ferramentas no mercado de desenvolvimento de software.

### 2. 🟡 Visualização e Modelos Mentais
Entender a árvore de commits pode ser um desafio abstrato no início. Estas ferramentas e explicações são excelentes aliados para demonstrar o comportamento do código "por baixo do capô".
* 🛠️ **[Visualizing Git](https://git-school.github.io/visualizing-git/)**: Ferramenta interativa fantástica para simular comandos no *browser* e ver as ramificações a crescerem em tempo real.
* 🎥 **[Como o Git funciona? (Série Sobrevivência #01)](https://www.youtube.com/watch?v=BAmvmaKQklQ)**: Uma exploração profunda do funcionamento interno do Git, mostrando como as árvores de diretórios, *blobs* e *hashes* se relacionam a cada *commit*.
* 🎥 **[Entender o Git de forma Visual](https://www.youtube.com/watch?v=v9ZM2PVzctM)**: Demonstração prática do modelo mental do Git para desmistificar o histórico de alterações e compreender melhor o fluxo.

### 3. 🟠 Resolução de Problemas e Comandos Avançados
No dia a dia do desenvolvimento de software, os erros acontecem. Saber recuperar ou reestruturar um histórico de *commits* é o que distingue os melhores profissionais.
* 🚑 **[Dangit, Git! (Oh Shit, Git!)](https://dangitgit.com/)**: O guia salva-vidas do programador para resolver os conflitos e erros mais temidos do terminal.
* 📺 **[Websérie: Git e GitHub para Sobrevivência](https://www.alura.com.br/webseries/git-e-github-para-sobrevivencia)**: Uma série inteira focada nos cenários de stress do dia a dia, ensinando a resolver problemas reais em ambientes de trabalho.
* 🎥 **[5 Coisas que precisa de saber sobre Git](https://www.youtube.com/watch?v=BU-w2_Aae54)**: Exploração de comandos mais complexos como *Rebase*, *Cherry-pick*, os diferentes tipos de *Reset* e a grande utilidade do *Stash*.

### 4. 🔵 Alojamento e Documentação do Portefólio
A forma como apresentamos o código ao mundo tem um peso gigantesco, especialmente para evidenciar competências técnicas em projetos robustos.
* 📄 **[O que é o GitHub Pages?](https://docs.github.com/pt/pages/getting-started-with-github-pages/what-is-github-pages)**: A documentação oficial sobre como utilizar o repositório para alojar sites estáticos.
* 🎥 **[Hospedagem Gratuita com GitHub Pages](https://www.youtube.com/watch?v=4-tfJ-ZyA0Q)**: O passo a passo definitivo para colocar os teus projetos no ar de forma gratuita.
* 🎥 **[Diferentes maneiras de compartilhar seu projeto](https://cursos.alura.com.br/extra/alura-mais/github-diferentes-maneiras-de-compartilhar-seu-projeto-c2002)**: Dicas essenciais sobre como dar a melhor visibilidade possível às aplicações desenvolvidas.
* 🎥 **[O poder de um bom README](https://www.youtube.com/watch?v=MqoqPzjQyCY)**: Um guia de como utilizar *Markdown* para criar documentação profissional, apelativa, rica em *badges* e blocos de código formatados para apresentar soluções de arquitetura.
